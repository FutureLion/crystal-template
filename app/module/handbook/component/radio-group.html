单选框组

实例

使用方法

引入模块

        
            var CheckBox_Group = require("module/control/radio-group"); //根据组件位置，引入组件模块

参数

名称 类型 默认值 描述
value observable对象 null 基本输入框
validator 校验组件 null 对当前属性的校验
options 对象含有text,value的数组 null 复选框组信息,text表示显示的内容，value表示选中对应的值

实例化模块，并传入对应参数

        
            var  week = new Radio_Group({
                            value: this.week,
                            validator: this.validators.week,
                            options: [
                                {
                                    text:'周一',
                                    value:1
                                },
                                {
                                    text:'周二',
                                    value:2
                                },                   
                                {
                                    text:'周三',
                                    value:3
                                },
                                {
                                    text:'周四',
                                    value:4
                                },                    
                                {
                                    text:'周五',
                                    value:5
                                },
                                {
                                    text:'周六',
                                    value:6
                                },
                                {
                                    text:'周日',
                                    value:7
                                }
                            ]
                    })

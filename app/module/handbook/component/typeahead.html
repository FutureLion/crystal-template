使用方法

引入模块

        
                var AbstractTypeahead = require('module/control/abstract-typeahead'); //根据组件位置，引入组件模块

参数

名称 描述
text 当前选中显示内容
value 当前选中对应的值
validator 校验组件

实例化模块，并传入对应参数

        
            this.typeahead = new AbstractTypeahead({
                text: 'hello',
                value: 1
            });

绑定模版

        
            <!--  ko module: typeahead  --><!-- /ko -->

静态文本组件介绍

目的

当前我们需要显示某个纯文本内容的时候，比如团单名称

实例

      
<!-- ko module: text --><!-- /ko -->
      
      
      

 <script>
 
	var text = require("module/component/form/text");

	model = [
		function(){
			this.dealName = ko.observable("我是团单");
			this.text = new text({
				value: this.dealName
			});
		}
	];
 
 </script>

使用方法

引入模块

		
			var text = require("module/control/static"); //根据组件位置，引入组件模块
			var ko = require("knockout");

参数

名称 类型 默认值 描述
text observable对象 null 静态文本内容

注意:这里text的值，也可以是普通文本内容

实例化模块，并传入对应参数

		
			this.dealName = ko.observable("我是团单"); //也可以普通文本this.dealName = "我是团单"
			this.text = new text({
				text: this.dealName
			});

绑定模版

		
	      	<!-- ko module: text --><!-- /ko -->

1、安装工具

安装node或者iojs

安装gulp

		npm install gulp -g

2、下载实例

使用 git clone 官网项目（或直接 下载 解压）

		git clone https://github.com/youngjay/crystal-template.git

进入项目目录

 		cd crystal-template

开发目录结构一览 

		
|-- README.md                      
|-- app                  -- 存放本应用有关代码
|   |-- asset            -- 存放资源文件，包括stylus以及图片
|   |-- binding-handler  -- knockout的binding handlers
|   |-- module           -- 存放组件，这是开发业务逻辑的主要目录
|   `-- service          -- 各个组件需要用到的公用服务
|-- dist                 -- 编译后的文件
|-- gulpfile.js          -- gulp task
|-- index.html           -- get start页面 
|-- mock-server          -- 模拟后台返回的数据 
|-- node_modules         -- 存放nodejs的库
|-- package.json         -- nodejs 库的配置
|-- server-config.js     -- 模拟后台返回的数据
|-- server.js            -- get startserver的脚本
、-- vendor               -- 非nodejs的库

1、构建项目

安装依赖包

		npm install

编译

		gulp

3、浏览项目

启动server

node server.js

完成服务器启动后，打开浏览器访问 http://127.0.0.1:3000 即可预览项目效果。

4、文件监听

gulp watch

保持node server.js启动的命令行终端不关闭，打开新的命令行终端，执行gulp watch命令，即可进入文件监听。修改项目中的代码并保存，刷新页面，刷新页面就可以看到最新更新效果（后续会做页面自动刷新功能，来提升开发效果）。

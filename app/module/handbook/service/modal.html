弹出框

我们基于bootstrap的modal对话框，来自定义我们的弹出框

实例

使用方法

引入模块

由于每个模块的实现都继承了vm.js的公共方法和属性，其中就有modal方法，因此不需要我们自己引用，框架已经自己引用了。

我们可以直接调用对应的modal方法this.modal

参数

名称 类型 描述
data object 数据传递
target 弹出框模块 弹出框对应的页面,该页面打开会立即执行onOpen方法，参数有data、callback,分别表示传过来的数据,自身的回调处理，感兴趣研究的具体参见/service/modal.js中的onOpen方法
confirm function 确认回调点击确认按钮触发
cancel function 取消回调，点击取消按钮触发
complete function 完成回调，弹出框完全关闭后会触发

实例化模块，并传入对应参数

        
            this.showModal=function() {
                    target: require('module/example/modal/modal-confirm'),
                    confirm: function(isOK) {
                        alert('I am ' + (isOK ? 'OK' : 'not OK'));
                    },
                    cancel: function() {
                        alert('I am not sure')
                    },
                    complete: function() {
                        alert('Bye')
                    }
                });

绑定事件

        
            <button data-bind="click: showModal">modal</button>

通知/notification

常用于信息提示（info）、成功提示（success）、警告提示（warning）、失败提示（error）

实例

使用方法

 		
 				//引入通知库
  				var notification = require("service/notification");

  				//调用对应通知方式 info、success、warning、error
  				notification.success("success提示内容", "信息提示");

相关接口

名称 类型 入参 描述
success/info/warning/error 函数 message、title、options 展现不同类型信息提示
clear 函数 无 清除当前信息提示

信息提示接口入参

success/error/info/warning(message, title, options)
名称 类型 默认值 描述
message 字符串 无 提示的信息内容
title 字符串 无 提示的标题
options object 内容较多，不一一列举，大致是一些动画效果，参考toastr 相关配置信息
